# Troubleshooting Ticket Examples

This folder contains realistic troubleshooting ticket examples for the `instant-dev-environment-lab` portfolio project.

These examples are written for IT Support, Technical Support, Service Desk, Desktop Support, and Junior Systems/Network Administration portfolio use.

## Tickets Included

| File | Topic |
|---|---|
| `codespace-startup-failure-ticket.md` | GitHub Codespaces startup issue |
| `devcontainer-build-error-ticket.md` | VS Code Dev Container build issue |
| `permission-denied-script-ticket.md` | Linux script permission issue |
| `node-port-conflict-ticket.md` | Node.js local port conflict |
| `git-authentication-issue-ticket.md` | GitHub/Git authentication issue |

## Portfolio Purpose

These ticket examples demonstrate troubleshooting structure, documentation habits, command-line awareness, and clear communication without using fake customer or company information.
