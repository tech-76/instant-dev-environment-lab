# Sample Projects

This folder contains beginner-friendly sample projects for the `instant-dev-environment-lab` repository.

## Projects Included

| Project | Purpose |
|---|---|
| `html-css-site` | Simple IT support landing page using HTML and CSS |
| `node-api` | Simple Express API with health check and sample ticket route |
| `python-script` | Python script that prints environment information and runs basic checks |

## Portfolio Purpose

These projects demonstrate practical beginner skills in web structure, API basics, scripting, documentation, and troubleshooting.
