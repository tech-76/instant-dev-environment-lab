# HTML/CSS Site

## What This Project Does

This is a simple IT support landing page built with HTML and CSS.

It includes:

- A professional hero section
- Example IT support service areas
- A basic support process section
- A sample support request form layout

The form is only a front-end example and does not submit real tickets.

---

## How to Run It

Open `index.html` in a browser.

If using VS Code, you can also use a local preview extension or right-click the file and open it in your browser.

---

## Skills Demonstrated

This sample project demonstrates:

- Basic HTML page structure
- CSS styling and layout
- Responsive design basics
- Clear service-focused writing
- Portfolio-ready front-end organization
