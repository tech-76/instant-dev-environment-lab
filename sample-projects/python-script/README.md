# Python Script

## What This Project Does

This Python script prints basic system and environment information, then performs simple troubleshooting checks.

It checks:

- Python version
- Operating system information
- Current folder
- Common command availability
- DNS resolution for GitHub
- Basic curl connectivity to GitHub

---

## How to Run It

From this folder, run:

```bash
python3 main.py
```

You can also run it from the repository root:

```bash
python3 sample-projects/python-script/main.py
```

---

## Skills Demonstrated

This sample project demonstrates:

- Running Python from the terminal
- Reading system information
- Checking whether commands exist
- Performing a basic DNS lookup
- Running a safe external command with Python
- Writing beginner-friendly troubleshooting output
